# ロゴ画像の配置について

ScoreFlowのロゴ画像を以下の場所に配置してください：

## ファイル名と場所
- ファイル名: `logo.png`
- 配置場所: `public/logo.png`

## 画像の要件
- 推奨サイズ: 512x512px または 1024x1024px（正方形）
- 形式: PNG（透明背景推奨）
- ファイルサイズ: 100KB以下を推奨

## 使用方法
画像を配置すると、以下の場所で自動的に使用されます：
1. サイトのファビコン（ブラウザのタブに表示）
2. LPページのヘッダー
3. ログイン後のホームページのヘッダー
4. 編集モード画面のヘッダー
5. 公開URLページのヘッダー

## 注意事項
- 画像ファイルを配置した後、開発サーバーを再起動してください
- Vercelにデプロイする際も、画像ファイルが含まれていることを確認してください
