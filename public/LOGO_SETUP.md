# ロゴ画像の配置手順

## 問題
現在、`public/logo.png`が存在しないため、ロゴ画像が表示されていません。

## 解決方法

### 1. 画像ファイルを準備
添付されたScoreFlowのロゴ画像を準備してください。

### 2. 画像ファイルを配置
以下の場所に画像ファイルを配置してください：

**ファイルパス**: `public/logo.png`

### 3. 画像の要件
- **ファイル名**: `logo.png`（必ずこの名前で）
- **推奨サイズ**: 512x512px または 1024x1024px（正方形）
- **形式**: PNG（透明背景推奨）
- **ファイルサイズ**: 100KB以下を推奨

### 4. 配置後の確認
1. ファイルを配置した後、開発サーバーを再起動してください：
   ```bash
   npm run dev
   ```

2. ブラウザで以下を確認：
   - ブラウザのタブにファビコンが表示される
   - 各ページのヘッダーにロゴが表示される

### 5. Vercelへのデプロイ
画像ファイルを配置した後、GitHubにプッシュすると、Vercelにも自動的にデプロイされます。

## 現在の状態
- コードは実装済み（画像が存在しない場合でもエラーは発生しません）
- 画像ファイルを配置すれば、すぐに表示されます
